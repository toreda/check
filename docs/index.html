@toreda/chk

Automated object and data validation using schemas.

 

Install

@toreda/chk is available as an NPM package.

 

Install with yarn:

yarn add @toreda/chk

or Install with NPM:

npm install @toreda/chk

Usage

This package uses Fate objects from @toreda/fate to return validation results.
