Class Tracer

Create a readable string to uniquely identify element instances based on their position in the schema tree relative to the root schema.

Example

const tracer = new Tracer();

Remark

Property name and parent element aren't specific enough to uniquely identify elements in recursively verified schemas, which may contain multiple sub-schemas with the same element names. e.g. schemaA.text and schemaA.schemaA.text would both look similar without the full path.
