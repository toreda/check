# `@toreda/chk`
Automated object and data validation using schemas.

&nbsp;


# Install
`@toreda/chk` is available as an [NPM package](https://www.npmjs.com/package/@toreda/chk).

&nbsp;

**Install with yarn:**
```bash
yarn add @toreda/chk
```

**or Install with NPM:**
```bash
npm install @toreda/chk
```
